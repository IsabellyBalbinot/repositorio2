# Sistema de Cadastro de peso e Alunos

Este projeto é um programa que solicita a entrada da idade, peso e altura de alunos. Ele calcula a soma de todos os números inteiros de 1 até a idade informada. O processo continua até que o usuário insira zero para a idade, encerrando o programa.
(exemplo.png)

## Descrição

O programa é escrito em uma linguagem de pseudocódigo e funciona da seguinte maneira:
- Solicita ao usuário que insira a idade, peso e altura de um aluno.
- Se a idade inserida for diferente de zero, o programa calcula e exibe a soma de todos os números inteiros de 1 até a idade inserida.
- O processo continua solicitando novos dados até que o usuário digite zero para a idade, o que finaliza o programa.
